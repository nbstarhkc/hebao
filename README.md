# GROUND ZERO // 3D 核爆自由沙盘

一个可直接部署到 GitHub Pages 的 WebGL 三维自由沙盘。爆炸、带炽白实体核心的热辐射火球、球形压力前沿、凝结壳、实体烟团蘑菇云、尘浪与音效都在浏览器中实时生成，不播放固定视频。火球由沸腾外层、高温内层和连续冷却的实体核心组成，不是透明空壳。

声音使用 Web Audio 实时合成：首次爆破会自动解锁声音，顶部扬声器也可播放短促低音确认。HRTF 空间定位跟随自由相机，地震耦合声先抵达、主声浪再按观察距离延迟抵达，并叠加压力裂响、低频冲击、长尾轰鸣、建筑碎裂、环境回声和持续风声。建议佩戴耳机体验。

性能管线会根据硬件自动启用 WebGL2 批处理、动态分辨率和最多 3 个后台 Worker：一个处理刚体物理，两个并行准备大当量爆炸的建筑影响范围。碎石采用实例化渲染，连续 10 Gt 模式会限制高成本刚体数量，其余建筑继续以批量实例形变显示破坏效果，避免运行时间越长越卡。

三维城市使用 Cannon 刚体物理和 9.81 m/s² 重力：建筑拥有独立质量、惯性、摩擦、碰撞体与睡眠状态。球形冲击波按三维距离抵达后施加径向脉冲，楼体可互相碰撞、倾倒、滑动并剥落带物理碰撞的碎块；完整楼体不会被当作轻质碎片抛上高空。

通过 HTTP 启动时，刚体世界自动放入独立 Web Worker，和主线程的 WebGL 渲染、烟团动画并行执行。大当量采用“近区刚体 + 远区实例化损伤”的物理 LOD，避免把全城数万栋建筑同时变成碰撞体。

## 启动

Three.js 与 Cannon 使用固定版本的 CDN 依赖。要启用多核物理，请在本目录启动静态文件服务器：

```powershell
npx serve .
```

然后打开终端显示的本地 `http://` 地址。页面顶部出现 `MULTICORE` 即表示物理 Worker 已启用。

如果还需要本地安装依赖进行开发或语法检查，可运行：

```powershell
npm install
```

## 自由沙盘操作

- 单击三维地面：选择任意爆点。
- `Space` 或红色按钮：立即投放，可无限叠加。
- 连续爆破：开启开关、设置间隔，再点红色按钮启动或停止。
- 按住鼠标拖动：自由环绕并可越过地平线仰视高空爆点。
- 鼠标滚轮：拉近或拉远。
- `WASD`：移动观察位置；`Q/E`：下降或上升；`Shift`：加速移动。
- `P`：暂停物理与爆炸时间；`C`：清空并重建沙盘。
- 顶部扬声器按钮：开启或关闭整个沉浸声场；暂停时音频时间也会冻结。
- 手机：点按地面选爆点、单指滑动环视、双指捏合缩放；左下摇杆移动，右侧按钮控制升降、引爆和参数面板。
- 手机顶部提供“电脑版 / 手机版”切换，选择保存在浏览器中；电脑版按完整桌面宽度缩放展示。
- 当量范围：10 kt 到 1 Tt（原上限的 100 倍）。
- 场景统一采用 `1 单位 = 1 米`；地图边长约 21 km，九个核心城区与实例化城市带铺满全图。
- 全图城市按 4 km 空间块进行视锥与距离裁剪；看不见的城区不会提交给 GPU。
- 10 Gt 自动使用极端当量 LOD：近区有限刚体、远区实例损伤、实体烟团隔帧更新、低填充率渲染和活跃特效预算。
- 渲染会根据帧率自动调整分辨率；调整发生在绘制前，不会产生黑色空帧。

本项目是游戏化视觉、声学与刚体物理模拟，界面读数不用于现实工程计算。
